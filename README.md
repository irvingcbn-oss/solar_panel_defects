# solar_panel_defects
This project used the pre-trained CNN ResNet50 to classify solar panel images. 
The script above performs image preprocessing and uses transfer learning whit fine-tuning in order to preserve the previously trained features of the ResNet50.

## Technologies
- Tensorflow
- Matplotlib
- Keras
- Numpy

## Dataset
https://www.kaggle.com/datasets/alicjalena/pv-panel-defect-dataset
